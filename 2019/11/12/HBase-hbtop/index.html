HBase 集成 hbtop

HBase 2.1.0-cdh6.3.1 体验下 hbtop 新功能

前言

HBase 2.1.0-cdh6.3.1 想体验下 hbtop 新功能。 hbtop 的介绍，从介绍可以看到目前 hbtop 不支持 HBase 1.x

实操

从 github 上下载 HBase 2.1.0-cdh6.3.1源码, Apache HBase 2.1.7源码

把 Apache HBase 的 hbase-hbtop moudle 拷贝到 CDH HBase 里。

在 CDH HBase 的 pom.xml 添加

修改 maven setting 文件, 记得修改 localRepository 为你本地的仓库地址

maven 加载依赖过程很长。等待着把。我花了7个小时才弄完。。。

依赖弄完后，pom 有些无关紧要的错误，忽略就好

进入 hbase-hbtop 项目下执行 maven 打包命令

当然也可以用 IDEA 来打包

进入 hbase-hbtop 项目 target 目录下可以看到已经打好的 jar 包

现在可以尝试部署到HBase集群上了，为了验证，先只在一台机器上做，成功后再每台机器上重复，当然，只在一台机器上部署也行，每次排查问题时都到这台
机器上操作即可

将 hbase-hbtop-2.1.0-cdh6.3.1.jar 拷贝到 /opt/cloudera/parcels/CDH/lib/hbase/lib 目录下

修改 bin/hbase

添加位置如图所示629行～646行

接着在conf目录下添加一个log4j-hbtop.properties 文件，用来设置 hbtop 的日志级别

在终端启动hbtop， hbase hbtop
